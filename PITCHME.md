### Hvorfor Kotlin?
@snapend

* Fungerer godt med Java
  * Java -> Kotlin
  * Kotlin -> Java
* Blir mye mindre kode for å oppnå det samme
  * *Rough estimates indicate approximately a 40% cut in the number of lines of code* (JetBrains)
* Nyttige features som:
  * Null safety
  * Extension functions
  * Data classes
* God verktøy støtte
